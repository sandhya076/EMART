# ECOMMERCE APPLICATION
This is a ECOMMERCE application with name of TRADITIONAL EMART which sells the TRADITIONAL clothes for both men and women.

## Packages used in this application
    javascript
    React
    Material-ui
    Redux
    Node
    Express
    JWT authentication
    Hasshing Password
    Stripe for payment
    Firebase for storage
    Emailjs
